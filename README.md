# Aman Prabhune — Portfolio

A modern, dark-themed single-page portfolio website for **Aman Prabhune** — Full-Stack Engineer & AI Systems Builder, currently pursuing an MISM at Carnegie Mellon University.

🔗 **Live Site:** [https://aman-1503.github.io/portfolio](https://aman-1503.github.io/portfolio)

---

## ✨ Features

- Dark glassmorphism design with glowing blue accents
- Animated particle canvas in the hero section
- Typewriter effect cycling through roles
- Smooth scroll with active navbar highlighting
- Fade-in animations on scroll (Intersection Observer)
- Mobile-responsive with hamburger menu
- Timeline layout for experience
- Devicon skill logos grouped by category
- Profile photo with circular CSS fallback ("AP" initials)

---

## 🖼️ Adding Your Profile Photo

1. Add a photo named **`profile.jpg`** to the **root** of this repository (same folder as `index.html`).
2. The image will be displayed as a circular photo with a glowing blue ring.
3. If `profile.jpg` is not found, the site automatically shows a circular fallback div with **"AP"** initials in blue.

> Recommended: square image, at least 400×400 px.

---

## 🚀 Deploy to GitHub Pages

1. Go to your repository on GitHub: `https://github.com/aman-1503/portfolio`
2. Click **Settings** → **Pages** (in the left sidebar)
3. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **Save**
5. After a minute, your site will be live at:

   **[https://aman-1503.github.io/portfolio](https://aman-1503.github.io/portfolio)**

---

## 🛠️ Local Development

No build tools needed — just open `index.html` in your browser:

```bash
# Option 1: open directly
open index.html

# Option 2: serve locally (Python)
python3 -m http.server 8080
# then visit http://localhost:8080
```
